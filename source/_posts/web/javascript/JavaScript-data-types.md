JavaScript 中：

- 数据类型：`number`、`string`、`boolean`、`null`、`undefined`、`symbol`、`bigint`、`object`
- 基本数据类型：`number`、`string`、`boolean`、`null`、`undefined`、`symbol`、`bigint`
- 引用数据类型：`object`

基本数据类型的特性是值无法修改，比如：修改字符串会得到一个新的字符串。

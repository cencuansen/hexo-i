| 类型        | 类型                                     |
| ----------- | ---------------------------------------- |
| String      | 字符串                                   |
| Hash        | 散列                                     |
| List        | 列表                                     |
| Set         | 集合                                     |
| Sorted Set  | 有序集合                                 |
| Bitmap      | 布隆过滤器                               |
| GeoHash     | 存储地理位置信息，并对存储的信息进行操作 |
| HyperLogLog | 用来做基数统计的算法                     |
| Streams     | 内存版 Kafka，消息订阅发布               |
